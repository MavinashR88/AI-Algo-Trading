Core Features

🌍

Dual-Market Trading

US markets via Alpaca (NYSE/NASDAQ) and Indian markets via Zerodha Kite Connect (NSE/BSE) running simultaneously 24/7.

🧠

AI Sentiment Analysis

Claude API scores news headlines on a -1.0 to +1.0 scale. Multi-source aggregation from NewsAPI, Google News RSS, and Yahoo Finance.

⚡

8 Trading Strategies

Momentum, mean reversion, breakout, MA crossover, RSI divergence, volume-price, sentiment, and hybrid ensemble voting.

🛡️

Psychology Guard

Anti-revenge trading, FOMO prevention, no averaging down, sentiment override checks. Simulates disciplined trader behavior.

📊

Risk Management

ATR-based position sizing, drawdown limits, trailing stops, configurable R:R ratios, and automatic market halts.

🔬

Backtesting Engine

Historical replay with Sharpe ratio, max drawdown, profit factor, win rate, and graduation criteria before going live.

📱

Live Dashboard

FastAPI + WebSocket real-time dashboard with halt/resume controls, trade history, P&L, and performance metrics.

🔐

Security Hardened

Fernet encryption for secrets, input validation, rate limiting, API key sanitization, and parameterized SQL queries.

💰

Tax Reporting

CSV exports with short/long-term classification, wash sale detection, and per-asset P&L summaries for both US and India.
